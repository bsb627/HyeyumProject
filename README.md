## 개요
+ 코로나 사태로 타격을 받은 문화 산업 분야와 개인의 문화 생활의 상생을 위한 새로운 형태의 기부 플랫폼
------------

## 설계의 주안점
- 기존의 직접 기부형태와는 달리 개인의 문화 활동 후 리뷰를 작성하면 기부금액으로 집계되도록 하여 개인의 문화활동과 동시에 문화산업에도 선한 영향력을 미칠 수 있도록 하는 데에 주안점을 두었다.
